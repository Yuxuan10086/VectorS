1. 机器人功能与参数

移动作业机器人：车载计算运行整机软件，移动平台负责行驶定位，SCARA 作业臂完成取放，电气系统统一供电并保护。底盘与机械臂共用 robot_driver 中的 CAN 电机栈。

车载计算

运行导航、感知与机械臂等全部 ROS 节点。

硬件平台Centerm 瘦客户端
处理器Intel J1900
内存 / 存储4 GB / 32 GB
操作系统Ubuntu 22.04 Server
中间件ROS 2 Humble
机载网络192.168.66.66，主机名 robot

移动平台

承担场地内行驶与停靠；与作业臂固定在同一机架上。

机架铝型材，约 450 mm × 450 mm
驱动前驱差速
后轮从动全向轮

SCARA 作业臂

安装在移动平台上，面向货架完成底层取放。

构型三轴 SCARA，工作高度约 350 mm
Z 轴丝杆
平面关节同步带传动

驱动与通信

移动平台与机械臂共 5 台正点原子 PD42 步进闭环，FOC，CAN 扩展帧；协议与接口见 robot_driver/motor。

电气与安全

为底盘电机、机械臂电机及主控分配电源，并具备独立保护与急停。

动力电池24 V，2550 mAh
工作电流持续约 6 A，峰值约 10 A
电机支路每路 5 A 独立保险丝
主控供电电池经降压模块供电
安全急停开关

4. 技术方案

TRT 机器人实验室 参赛机器人：固定路线粗定位 → 激光精调 → 视觉精调 → 机械自适应兜底。物块贴 AprilTag，仅作业底层。

固定路线到站 → IMU + 底盘 SDK 底层闭环 → 激光：距离 + 平行度 → 单目 AprilTag：横向 → 机械自适应抓取
层级说明
路线 沿预设固定路线行驶，到达面向目标物块的停靠位。
底盘底层 IMU 与底盘 SDK 融合，只参与底层运动控制：根据实际运动与速度指令的误差做修正，不承担全局规划。
激光 二维激光调整与柜台/货架的距离与平行度。
视觉 单目相机、无深度；识别 AprilTag，解算物块位姿，修正横向位置。
机械 SCARA 与机构自适应为最后一层兜底，吸收残余误差。

数据流：cmd_vel 经底盘闭环执行 → 到点后激光对齐柜面 → 相机对 tag 横向微调 → 机械臂抓取或放置。

5.2 立体结构与取货层

带围挡桌面场地，地面可通行。平面黄/蓝区块为双层货架。

底层 ≈25–30 cm 高层 ≈65–75 cm

货架侧视剖面

单元结构：B/C 长货架、A/D 同类

  • 底层：浅蓝基座台面，约 25–30 cm
  • 高层：白立柱 + 白台面，约 65–75 cm
  • 立柱为障碍，进通道须留横向余量
本队取货层：末端有效高度 ≤400 mm，仅做底层取货与 A 区底层放置。
区域立体特征导航/作业要点
B / C 商品区 两座平行长货架 200×40 cm，间距 130 cm 成巷道；底层格位 1–6 沿长度排列。 驶入 B–C 巷道或贴外沿；对准底层格位驻车，侧面伸臂，避立柱。底盘约 450 mm。
A 出货区 左下方形双层架；底层为放置区。 物块落在底层台面内；预留臂展，避立柱。
D 挑战区 右上黄色基座 + 白色框架；前方地面橙色标识为停靠参考。 可选任务；建议从场地右侧开阔区绕入，橙标前停车，底层拿货后回 A。
E / F 左上 E 50×50 cm 起点，左中 F 50×50 cm 停放。 E 发车；结束须入 F。
主通道 3D 中紫色地面为巡航区；C 区以东右侧空地大，适合掉头与 D→A 转运。 建议宽通道巡航 + 巷道精确入位；激光需关注立柱与上层台面下沿。

导航阶段

  1. 巡航：E → 主通道或 B–C 巷道口
  2. 对接：低速进巷道，按格位 1–6 驻车
  3. 作业：驻车抓取/放置，避免整车前进撞柱。
  4. 回运：退出巷道 → 主通道至 A → 可选 D → 入 F。

5.3 任务流程

  • 赛前声明 自主 / 人工操控；自主运行不得人工操控。
  • 赛前抽 2 张取货号；区内任选格位抓取。
  • 2 块物块全部入 A 区；可选 D 区 拿货至 A，+20 分。
  • 全部任务结束后机器人须进入 F 区，否则扣分；抓取时尽量避免碰撞其它物体。
  • 自主设备启动后不得干预系统整体控制。

5.4 时间与计分

  • 入场后、运行前有 40 分钟调试；取最优一次成绩。
  • 每次运行 10 分钟，从设备开始运行计时；选手示意完成或到时为一次。
  • 技术分 80 + 报告分 20。

技术分细则

  • 拿货 +20/件，入 A 放置 +20/件，合计上限 80；出界 −10/件。
  • D 区成功 +20，总分仍封顶 80。
  • 碰撞、跌落、破坏场地、人工干预：每次 −5。
  • 结束未进 F 区 −5。
  • 系数：自主控制×导航 1.5；算法完成 1.2；有人工介入 1.0；全程人工 0.75。

报告分须含六部分正文与运行视频；细则以赛项群为准。
